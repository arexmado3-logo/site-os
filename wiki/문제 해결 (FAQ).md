# 문제 해결 (FAQ)

## Q1. 공유 코드를 넣었는데 가져오기가 실패해요.
- 코드가 중간에 잘리지 않았는지 확인하세요.
- 앞뒤 공백/줄바꿈이 섞인 경우 제거 후 다시 시도하세요.

## Q2. 팝업에서 복사 버튼이 동작하지 않아요.
- 브라우저 권한 정책 또는 클립보드 접근 상태를 확인하세요.
- 확장을 새로고침 후 다시 시도해 보세요.

## Q3. 데이터가 안 보이거나 초기화된 것 같아요.
- 다른 크롬 프로필로 실행 중인지 확인하세요.
- 동기화 설정과 계정 로그인을 확인하세요.

## Q4. 미리보기 페이지가 비어 있어요.
- 유효하지 않은 공유 코드일 수 있습니다.
- 새로 생성한 공유 코드로 다시 열어보세요.


## 문의
문제 해결이 안 되면 문제점에 문의하세요.
