# 설치 가이드

## 요구 사항
- Google Chrome 최신 버전

## 설치 절차
1. 저장소를 로컬에 준비합니다.
2. Chrome 주소창에 `chrome://extensions` 입력
3. 우측 상단 **개발자 모드** 활성화
4. **압축해제된 확장 프로그램 로드** 클릭
5. 프로젝트 폴더 선택

설치가 완료되면 툴바에서 Site OS 아이콘이 보입니다.

## 업데이트 방법
코드를 변경한 뒤 `chrome://extensions`에서 해당 확장의 **새로고침** 버튼을 클릭하면 반영됩니다.
