# Site OS Wiki

Site OS는 크롬 확장으로 여러 개의 개인 사이트를 저장하고 공유 코드로 다른 사용자와 공유할 수 있는 프로젝트입니다.

## 빠른 시작
1. Chrome에서 `chrome://extensions`를 엽니다.
2. **개발자 모드**를 활성화합니다.
3. **압축해제된 확장 프로그램 로드**를 눌러 이 저장소 폴더를 선택합니다.
4. 확장 아이콘을 클릭하고 사이트를 생성합니다.

## 주요 기능
- 사이트 여러 개 생성 및 저장
- 저장된 사이트 목록 관리 (미리보기/삭제)
- 공유 코드 생성 및 다른 사용자 가져오기
- 뷰어 페이지(`viewer.html`)로 공유 콘텐츠 확인
