# 사용자 가이드

## 1) 새 사이트 만들기
1. 확장 팝업을 엽니다.
2. **사이트 이름**과 **사이트 본문**을 입력합니다.
3. **저장** 버튼을 누릅니다.

## 2) 저장된 사이트 관리
- **미리보기**: `viewer.html` 화면으로 열어서 콘텐츠를 확인
- **공유코드**: 공유 가능한 코드를 클립보드에 복사
- **삭제**: 목록에서 사이트 제거

## 3) 공유 코드로 가져오기
1. 다른 사용자에게 받은 공유 코드를 복사합니다.
2. 팝업 하단의 입력창에 붙여넣습니다.
3. **가져오기** 버튼을 누르면 내 목록에 추가됩니다.

## 데이터 저장 위치
- 데이터는 `chrome.storage.sync`에 저장됩니다.
- 동일한 구글 계정 동기화 환경에서 확장 데이터가 동기화될 수 있습니다.
