# 공유 코드 동작 원리

Site OS 공유 코드는 서버 없이 텍스트 코드만으로 데이터를 전달하기 위한 방식입니다.

## 포맷
공유 코드는 아래 JSON 데이터를 문자열화한 뒤 Base64로 인코딩한 값입니다.

```json
{
  "name": "사이트 제목",
  "content": "사이트 본문"
}
```

## 흐름
1. 사용자가 **공유코드** 버튼 클릭
2. 선택한 사이트의 `name`, `content`를 인코딩
3. 생성된 코드를 클립보드로 복사
4. 수신자는 코드를 붙여넣어 디코딩 후 가져오기

## 주의
- 공유 코드는 암호화가 아니라 인코딩입니다.
- 민감한 정보는 공유 코드에 넣지 않는 것을 권장합니다.
